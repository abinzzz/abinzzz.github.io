3.线程池

线程池(Thread Pool)是一种线程管理的方法,可以有效控制线程的数量,提高系统的性能。

其基本思想是:

  1. 先创建指定数量的线程,将这些线程放入线程池中待命。

  2. 当有新的任务来时,从线程池中去除一个空闲的线程来执行任务,执行完毕后,该线程又回到线程池中待命。

  3. 如果所有线程都在工作,任务会被放到一个任务队列中等待,直到有线程空闲出来。

线程池的主要优势:

  • 重用存在的线程,减少线程创建和销毁的开销,提高响应速度。

  • 可以限制线程的最大数,提高系统资源的利用率,避免过多线程竞争资源。

  • 可以更好的管理线程,设置空闲线程的回收时间,自动调优线程数等。

  • 提供了异步执行任务的方式。

这个“attributeerror: nonetype 对象没有属性分割“错误信息。正如我们上面提到的，表明您正在尝试对值为 None 的变量或对象调用split() 方法。

简而言之，split() 方法只能在字符串对象上调用，但在这种情况下，您调用split() 方法的变量或对象是 None，这意味着它没有分配任何值到它。

主要原因是在获取OpenBLAS版本号时调用了openblas_get_config函数,但是这个函数在你的环境中找不到,返回了None。
后续在对返回的None调用split方法时报错 AttributeError: ‘NoneType’ object has no attribute ‘split’。

NoneType是什么

在Python中,NoneType是None对象的类型。

None对象在Python中表示什么都没有,相当于其他语言中的null或nil。它是Python的特殊对象之一。

当您尝试对“none”值调用split() 方法时，Python 会引发“nonetype”对象没有属性“split” 错误是因为“none”没有split() 方法。

  • 创建方式
  • 查看数据类型


  • NoneType与None对象的主要特点

    • NoneType只有一个值——None。它是一个只有一个值的类。

    • NoneType的类型是type。type(None)返回NoneType。

    • NoneType是不可变的,无法创建其他NoneType的实例。

    • 使用==或者is判断一个对象是否是None时,实际上是判断类型和对象身份。

    • None对象通常用来表示无值或缺失值。

split()的作用是什么

Python 中的split () 方法用于根据指定的分隔符或定界符将字符串拆分为子字符串列表。
