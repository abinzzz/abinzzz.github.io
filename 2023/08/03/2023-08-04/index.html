RandomUnderSampler().fit_sample()

1.作用

  • 从多数类(majority class)中随机移除一些样本,以减少多数类样本的数量
  • 从而使训练数据集中不同类别样本的比例更加平衡
  • 通常用来处理分类问题中的类别不平衡(class imbalance)问题

2.演示

假设我们正在训练一个模型来分类水果,有两个类别:

  • 苹果(类别0) - 1000个样本
  • 橘子(类别1) - 100个样本

很显然类别不平衡,苹果的样本数远多于橘子。为了平衡训练数据,我们可以使用RandomUnderSampler:

rus.fit_resample()会从苹果的1000个样本中随机删除一部分,使苹果和橘子的样本数量接近。

假设它抽取了苹果到100个样本,那么我们现在的训练数据是:

  • 苹果(类别0) - 100个样本
  • 橘子(类别1) - 100个样本

样本数量变得平衡!我们可以用这些平衡后的数据训练模型,从而提高模型对少数类橘子的预测能力。

测试数据保持原始比例不变。这样模型在实际使用时,仍能对真实世界中类别不平衡的数据进行预测。

SMOTE(‘minority’).fit_resample(X_train, y_train)

使用SMOTE算法对训练数据进行过采样。

SMOTE(Synthetic Minority Over-sampling Technique)是处理分类问题中类别不平衡的数据的一种方法。

1.工作原理

  • 找到少数类样本
  • 对于少数类的每个样本,计算其k个最近邻(通常k=5)
  • 随机选择几个邻居样本,并连接当前样本与其邻居样本生成新的合成样本

2.参数

这里的 ‘minority’ 表示仅对少数类进行过采样,以平衡各类别样本数量。

fit_resample() 方法将SMOTE应用到训练集X_train和y_train上

3.作用

这样通过合成新样本的方式可以平衡各类别样本量,防止分类模型向多数类偏斜,提升对少数类的识别能力。

macro avg、micro avg 和 weighted avg

区别

macro avg、micro avg 和 weighted avg 是计算多类别分类问题评估指标的不同方法。它们的主要区别如下:

  • macro avg:先计算每个类别的指标,再取所有类别指标的无权平均值(给各类别指标相同权重)。
  • micro avg:将所有样本视为一个整体,直接计算总体指标(权重和样本数成正比)。
  • weighted avg:计算每个类别的指标,再取所有类别指标的加权平均值（权重和每个类别的样本数成正比）。

举例

假设二分类问题,类别A有100样本,类别B有20样本。对应的准确率分别是0.9和0.8。

则:

  • macro avg = (0.9 + 0.8) / 2 = 0.85
  • micro avg = (0.9 100 + 0.8 20) / (100 + 20) = 0.88
  • weighted avg = (0.9 100 + 0.8 20) / (100 + 20) = 0.88

可以看出,macro avg treat所有类别均等,而后两种权重和样本量相关。

Dense()

在Keras中,Dense层是最基本的全连接网络层。它可以实现输入和输出之间的全连接

参数

  • units:该层的输出维度,即输出特征数。比如64。

  • use_bias:是否使用偏置项(bias),默认True。
  • kernel_initializer:权重矩阵初始化函数,如’glorot_uniform’等。
  • bias_initializer:偏置向量的初始化函数。

Dropout()

在Keras中,Dropout()层用于防止神经网络过拟合。

在训练过程中,Dropout层每次更新参数时会随机令一定比例的节点输出值为0,在测试时使用整个网络。

主要参数

  • rate: 需要断开的连接的比例,断开比例越大,正则化效果越明显,默认0.25
  • noise_shape: 生成的随机数的维度,用于控制从哪些维度断开连接,默认为输入张量的维度值
  • seed:随机种子,用于产生随机数,默认None




model.compile()

在Keras中,model.compile() 用来配置模型的学习过程

作用

  • 指定模型优化器(optimizer):如’adam’,’sgd’等,控制权重更新的算法。
  • 指定损失函数(loss):如’mean_squared_error’, ‘categorical_crossentropy’等,评估模型训练效果。
  • 指定评估指标(metrics):如’accuracy’等,用于监控训练过程中模型的表现。
