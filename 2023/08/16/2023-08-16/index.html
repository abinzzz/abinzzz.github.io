1.xgboost.DMatrix()

xgboost.DMatrix() 是XGBoost中的一个核心数据结构,它是用于封装和处理训练数据的

主要作用

  • 封装特征和标签
  • 处理洗漱数据

  • 提供额外信息

  • 统一数据接口：几乎所有XGBoost都支持DMatrix作为输入

xgboost.DMatrix()是一个非常重要的API,我们通常需要先构造DMatrix对象,然后作为数据集输入给XGBoost模型的各种训练、预测方法。它为XGBoost提供了一个优化的、统一的数据接口

2.model = xgboost.train()

xgboost.train()是XGBoost中用于训练模型的核心函数。它实现了XGBoost的提升树算法,可以高效地进行提升树模型的训练。

主要参数

  • params:字典形式的训练参数,包括学习率、最大树深度、叶子节点数等 hyperparameters。
  • dtrain:训练数据,通常是xgboost.DMatrix对象。
  • num_boost_round:提升迭代的次数,即模型包含树的数量。
  • evals:评估数据集列表,用于监控训练进度。
  • obj:自定义的优化目标函数。
  • feval:自定义的评估函数。
  • xgb_model:可选的,用于增量训练的已有模型。

4.model.get_dump()

在XGBoost中,model.get_dump()函数可以输出模型的详细信息,用于诊断和分析模型。

主要作用

  • 打印树的结构信息：它会打印出每一棵树的节点分裂条件,树的形状,叶子节点的预测值等信息。这可以用来查看树的生长情况。

  • 查看特征的重要性：根据特征在树的分裂中的使用情况,可以汇总出特征的重要性打分。

  • 推断提升过程：打印每棵树的叶子节点预测值分布,可以帮助理解模型的提升过程。

  • 分析模型复杂度：如树的最大深度、叶子节点数等,帮助判断模型是否过拟合。

  • 检查模型在数据分布上的拟合效果：看预测值分布是否合理,是否有过大偏差等。

参数

  • with_stats:是否输出额外的统计信息,如特征重要性等。
  • dump_format:输出格式,支持’text’、’json’。

示例

13.model.coef_()

在机器学习模型中,coef_属性通常表示模型的参数权重系数。

对于线性回归和逻辑回归等线性模型来说,coef_保存了各个特征对应的模型权重值。

以逻辑回归为例,如果模型是:

logit(y) = w0 + w1x1 + w2x2 + … + wn*xn

其中y是预测的目标,x1到xn是输入特征,w0到wn是对应的权重系数。

那么在逻辑回归模型model中,model.coef_就是一个NumPy数组,包含了w0到wn这些权重值。

model.coef_.shape 是一个(n+1, )的向量,n是特征数。
model.coef_[0]对应w0,常数项权重
model.coef_[1:]对应w1到wn,各个特征权重
通过分析coef_中的权重大小和正负,可以判断不同特征对模型结果的贡献程度和方向。这在模型解释和特征分析中很有帮助。
