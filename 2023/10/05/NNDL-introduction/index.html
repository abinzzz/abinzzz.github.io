1.学习的一般分类

学习：系统从经验中提高性能的任何过程。

机器学习：

  • 有监督学习

  • 强化学习

1.1 机器学习介绍

机器学习：

  • 赋予计算机学习能力而无需显示编程的研究领域
  • 适定性学习问题：对于某类任务T和性能度量P，如果一个计算机程序在T上以P衡量的性能随着经验E而自我完善，那么称这个计算机程序在从经验E中学习。

例子：

  • T：下围棋
  • P：比赛中击败对手的百分比
  • E：历史比赛数据或和自己进行对弈

  • T：识别和分类图像中的手写文字
  • P：分类的正确率
  • E：已知类别的手写文字数据库

1.2 有监督学习

有监督学习：Supervised Learning

数据：D={d(1)d^{(1)}d(1),d(2)d^{(2)}d(2),…,d(n)d^(n)d(n)},n个样本

  • d(i)d^{(i)}d(i): ( x(i)x^{(i)}x(i),y(i)y^{(i)}y(i) )
  • x(i)x^{(i)}x(i):输入向量
  • y(i)y^{(i)}y(i):x(i)x^{(i)}x(i)对应的类别/输出

目标🎯：学习一个映射f(x -> y)

  • 回归：x离散或连续，y连续
  • 分类：x离散或连续，y离散


1.3 无监督学习

无监督学习：Unsupervised learning

数据：D={d(1)d^{(1)}d(1),d(2)d^{(2)}d(2),…,d(n)d^(n)d(n)},n个样本

  • d(i)d^{(i)}d(i): x(i)x^{(i)}x(i)
  • x(i)x^{(i)}x(i): 输入向量
  • 没有类别/输出y

目标：学习样本之间的关联，样本的成分

问题类型：

  • 聚类：将相似样本划分到同一簇
  • 密度估计：根据训练样本确定样本总体的概率分布



1.4 强化学习

强化学习：Reinforcement Learning

数据：历史时刻上的状态，动作以及回报

  • 没有y，取而代之的是一个判别学习器性能优劣的一个奖励信号

目标：学习一个使得长期积累回报最大的策略

  • π\piπ: S x A →\rightarrow→ P

例子：

人：

  • 状态：年龄，健康，…
  • 动作：读书，学拉丁舞，早恋,…
  • 回报：+100，-100
  • 目标：人生赢家

2.典型的机器学习过程

学习任务：学习器学习经验数据D，以实现

  • 对未知数据的预测
  • 对已知数据的描述

过程：

  • 选择一个模型
  • 选择一个目标函数
  • 选择一个学习算法

  • x:输入变量
  • y:输出变量
  • (x,y): 一个训练样本
  • (x(i)x^{(i)}x(i),y(i)y^{(i)}y(i)):第i个训练样本
  • n：训练集中训练样本的个数
  • x=(x1x_1x1​,…,xdx_dxd​),d为输入向量的维度

3.3 过拟合(ovefitting)

过拟合：模型对训练数据拟合程度很高,但对新数据的预测能力较差的情况

一味的最小化训练误差不是一个好的选择，更重要的是模型在未知数据上的表现如何

过拟合的原因：

  • 模型参数过多，自由度过大
  • 训练数据太少

如何避免过拟合：

  • 增加训练数据量：更多的训练数据可以让模型学习更多信息,减少过拟合风险。
  • 正则化：在模型中添加正则项,如L1、L2正则化,可以惩罚复杂的模型,使其不要过于依赖训练数据。
  • 早停：在训练过程中监控验证集误差,当验证集误差不再下降时提前停止训练。
  • 丢弃法：随机丢弃节点或者丢弃输入样本,这迫使模型去学习更稳健和泛化性更好的特征。
  • 批标准化：通过标准化每层输入的均值和标准差可以有效减少内部协变量移位。
  • 使用验证集： 将部分训练数据单独作为验证集,用于指导模型训练和超参数调优,避免完全依赖训练数据。
  • 使用L2权重衰减： 在模型权重参数上添加L2正则化项,可以惩罚较大的参数值。
  • 数据增强：通过对训练数据做一定变换(如旋转、剪切、缩放等)来增加数据多样性。
  • 采用流行的正则化网络结构,如Dropout等。

3.4 归纳偏置

归纳偏置：机器学习模型由于其本身结构或假设的限制,在进行归纳推理时存在的偏差

例如：

  • 线性回归模型假设目标函数为线性模型,这是它的偏置。对非线性问题进行建模时就存在偏差。
  • 感知机模型假设目标函数是线性可分的,这也是一种偏置。
  • 优化算法如梯度下降有假设目标函数是凸函数,不满足时就存在偏差。
  • 使用的训练集主要是猫的图片,模型就会偏向猫的特征

4.模型求解以及最优化

采用一个学习算法用以学习一个最优假设

两个问题：

  • 最优假设的标准是什么
  • 是否有必要求解最优解的精确值

4.1 两类误差

训练误差：1n\frac{1}{n}n1​ ∑i=1n\sum^{n}_{i=1}∑i=1n​ (y(i)−hw(x(i)))2(y^{(i)}-h_w(x^{(i)}))^2(y(i)−hw​(x(i)))2

真实误差/泛化误差：E(x,y)E_{(x,y)}E(x,y)​[(y(i)−hw(x(i)))2(y^{(i)}-h_w(x^{(i)}))^2(y(i)−hw​(x(i)))2]

往往需要训练误差逼近泛化误差，好的训练误差不一定意味着好的泛化误差，最优化训练误差可能会导致过拟合。


估计泛化误差：

  • 理论上：大数定律
  • 实际上：实验时从训练数据中抽取m个数据用于模型的测试

例子：线性回归

  • 假设：hwh_whw​(x)=w0w_0w0​ + w1w_1w1​x
  • 参数：w0w_0w0​,w1w_1w1​
  • 代价函数：Loss(y,hw(x)h_w(x)hw​(x))=1n\frac{1}{n}n1​ ∑i=1n\sum^{n}_{i=1}∑i=1n​ (y(i)−hw(x(i)))2(y^{(i)}-h_w(x^{(i)}))^2(y(i)−hw​(x(i)))2
  • 目标：min J(w0,w1w_0,w_1w0​,w1​)

4.3 求解方法2: 梯度下降法

基本思想：根据误差减少的方向调整权重

假设：hwh_whw​(x)=w0w_0w0​ + w1w_1w1​x

参数：w0w_0w0​,w1w_1w1​

代价函数：Loss(y,hw(x)h_w(x)hw​(x))=1n\frac{1}{n}n1​ ∑i=1n\sum^{n}_{i=1}∑i=1n​ (y(i)−hw(x(i)))2(y^{(i)}-h_w(x^{(i)}))^2(y(i)−hw​(x(i)))2

更新：wj′=wj−α∂∂wjJ(w0,w1)w_j^{'}=w_j - \alpha\frac{\partial}{\partial w_j}J(w_0,w_1)wj′​=wj​−α∂wj​∂​J(w0​,w1​) (α\alphaα是学习率)

学习率的选择：

  • α\alphaα太小，梯度下降将会缓慢地收敛
  • α\alphaα太大，损失函数可能不会每次迭代都减少，也可能不收敛

分类：

  • 批量梯度下降(BGD): 每一次迭代时使用所有样本来进行梯度的更新
  • 随机梯度下降(SGD): 每次迭代使用一个样本来对参数进行更新
  • 小批量梯度下降(MBGD): 折中的方法，每次迭代使用 batch_size 个样本来对参数进行更新
