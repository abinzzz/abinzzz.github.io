好的，这个分数项 1+e−β1−e−β\frac{1+e^{-\beta}}{1-e^{-\beta}}1−e−β1+e−β​ 是通过对单个帧内部所有空间位置的空间衰减因子进行求和，并利用无穷等比数列（infinite geometric series）求和公式计算出来的。

以下是详细的推导步骤：

第一步：构建求和表达式

我们的目标是计算在一个帧内，以某个参考点为中心，所有空间位置的衰减因子 e−βde^{-\beta d}e−βd 的总和（其中 d 是空间距离）。

这个总和可以表示为：
Sum = (\text{位置0的衰减}) + 2 \times (\text{所有其他位置的衰减之和})$

  • 位置0的衰减: 参考点自身的衰减，此时距离 d=0，所以衰减因子为 $e^{-\beta \cdot 0} = 1$。
  • 其他位置的衰减: 空间距离 d 可以向两个方向展开（左和右），所以我们用 2 乘以单边所有距离（d=1, 2, 3, \dots）的衰减因子之和。

因此，求和表达式为：
$1 + 2 \sum_{d=1}^{\infty} e^{-\beta d}$

第二步：识别并计算等比数列

后面的求和项 $\sum_{d=1}^{\infty} e^{-\beta d}$ 是一个无穷等比数列：
$e^{-\beta} + e^{-2\beta} + e^{-3\beta} + \dots$

  • 首项 (a): $e^{-\beta}$
  • 公比 ®: $e^{-\beta}$

应用无穷等比数列的求和公式 Sum = a / (1 - r)，我们得到：
$\sum_{d=1}^{\infty} e^{-\beta d} = \frac{e^{-\beta}}{1 - e^{-\beta}}$

第三步：合并并化简

现在，将第二步的结果代回到第一步的表达式中：
$1 + 2 \left( \frac{e^{-\beta}}{1 - e^{-\beta}} \right)$

为了化简，进行通分：
$= \frac{(1 - e^{-\beta}) + 2e^{-\beta}}{1 - e^{-\beta}}$

合并分子中的同类项：
$= \frac{1 + e^{-\beta}}{1 - e^{-\beta}}$

结论：
通过以上步骤，我们就得到了您想了解的这个分数项。它代表了在一个被完全忽略的帧中，所有空间位置的注意力衰-减因子的总和的上界。
