说在前面

本文主要内容参考 Claude Code - Full Tutorial for Beginners 和 Claude Code - Quick Start。

Quick Start

什么是 Claude Code？

  • 它是一个命令行应用程序，类似于 AI 代理，可以直接在终端中根据你的自然语言描述来编写代码、创建网站或调试项目。
  • 它是 Anthropic 官方推出的工具（类似于 Cursor 的终端增强版）。

前期准备与安装

  • 订阅要求：需要有 Claude Pro/Team/Enterprise 订阅，或者在 Claude 控制台有 API 余额。
  • 安装方式：
    • Mac/Linux：使用 curl 命令安装。
    • Windows：建议使用 PowerShell，通过 irr 命令安装。
  • 启动：在终端输入 claude 即可启动，首次使用需登录 Anthropic 账号进行身份验证。

最佳实践配置

为了发挥最大威力，建议配合以下工具：

  • Git：用于代码版本控制，方便在 AI 出错时回滚。
  • GitHub：通过 gh login 命令连接，让 Claude 自动创建仓库并上传代码。
  • IDE (编辑器)：建议配合 VS Code 或 Cursor 使用。在编辑器中打开终端运行 Claude Code，可以实时看到文件变化。

核心功能与操作技巧

  • 三种模式：
    • Ask (默认)：执行前会征求你的许可。
    • Code：自动接受修改并执行代码。
    • Planning：先制定详细计划，确认后再执行（适用于复杂项目）。
  • 快捷键：
    • Shift + Tab：循环切换模式。
    • Esc 连按两次：清空当前输入框。
    • Alt/Option + Enter：换行输入长内容。
    • /model：切换模型（Opus - 最强, Sonnet - 均衡, Haiku - 快速低成本）。

高级特性

  • 持久记忆 (claude.md)：通过 /init 初始化。这个 Markdown 文件充当 Claude 的“长期记忆”，你可以把项目的编码规范、特定规则写 in 里面，Claude 每次启动都会阅读它。
  • 后台任务 (/tasks)：可以管理长时间运行的任务（如本地服务器），不会阻塞对话。
  • MCP (模型上下文协议)：允许 Claude 连接 Notion、Gmail、Google Drive 等外部应用。
  • Agents & Skills：可以创建专门负责特定任务的子代理， or 教导 Claude 一些可重复使用的技能。
