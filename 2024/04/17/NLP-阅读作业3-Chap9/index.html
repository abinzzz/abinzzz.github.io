1.RNN

1.1 RNN基础

(1)基本结构

简单的循环神经网络在结构上类似于普通的前馈网络，输入向量表示当前输入xtx_txt​，通过权重矩阵乘以后，通过非线性激活函数计算隐藏单元层的值。然后使用这个隐藏层计算相应的输出yty_tyt​。与我们之前基于窗口的方法不同，序列通过逐个呈现一个项目给网络来处理。我们将使用下标来表示时间，因此xtx_txt​将表示时间t的输入向量。与前馈网络的主要区别在于图中的循环链接，这个链接通过增加隐藏层计算的输入来增加来自时间点前的隐藏层的值。

前一个时间步的隐藏层提供了一种记忆或上下文形式，编码了早期处理并告知稍后时间点的决策。关键的是，这种方法不对这种先前上下文施加固定长度的限制；前一个隐藏层中体现的上下文可以包括从序列开始到现在的所有信息。


(2)训练:

W:从输入层到隐藏层的权重
U:从前一个隐藏层到当前隐藏层的权重
V:从隐藏层到输出层的权重

突出显示了我们在前馈网络的反向传播中不必担心的两个考虑因素。首先，为了计算时间t的输出的损失函数，我们需要时间t − 1的隐藏层。其次，时间t的隐藏层既影响时间t的输出，也影响时间t + 1的隐藏层（因此影响t + 1的输出和损失）。由此可见，为了评估hth_tht​的错误，我们需要知道它对当前输出以及后续输出的影响。

为这种情况量身定做的反向传播算法导致了一种双向算法，用于训练RNN中的权重。在第一阶段，我们进行前向推理，计算hth_tht​、yty_tyt​，在时间的每一步累积损失，保存每一步的隐藏层值，以供下一时间步使用。在第二阶段，我们以相反的顺序处理序列，计算所需的梯度，计算并保存每一步向后时间的错误项。这种通用方法通常被称为通过时间的反向传播.





1.2 RNN作为语言模型

之前学过的NNLM模型的缺陷在于，输入数据要求是固定长度的，与N-gram类似，也假设当前词仅依赖于前n-1个词。语言模型任务是一个序列预测问题，而RNN是天然用来解决序列问题的模型，且在RNN模型中，理论上之前所有的单词都会影响到当前单词的预测。

对于当前时刻t，输入由两部分组成：当前位置单词的词向量xtx_txt​，以及上一时刻的隐状态ht−1h_{t-1}ht−1​，输出为下一个位置的单词预测概率分布y^t\hat y_ty^​t​，这个分布是基于整个词表|V|的，因此维度也是|V|。真实的分布是yty_tyt​，是一个ont-hot向量，在正确单词的位置上的值为1，其它位置的值为0。

隐藏层的特征表示为：

ht=σ(W(hh)ht−1+W(hx)xt)h_t=\sigma(W^{(hh)}h_{t-1}+W^{(hx)}x_t) ht​=σ(W(hh)ht−1​+W(hx)xt​)

输出的概率分布y^t\hat y_ty^​t​为：

y^t=softmax(W(S)ht)\hat y_t=softmax(W^{(S)}h_t) y^​t​=softmax(W(S)ht​)

其中的W(hh),W(hx),W(S)W^{(hh)}, W^{(hx)}, W^{(S)}W(hh),W(hx),W(S)都是参数矩阵，与时刻t无关，σ()\sigma()σ()是sigmoid激活函数。

模型的损失函数是交叉熵损失函数,对于正确单词的位置j，yt,j=1y_{t,j}=1yt,j​=1，其它位置都是0：

J(t)(θ)=−∑j=1∣V∣yt,j×log(y^t,j)J^{(t)}(\theta)=-\sum_{j=1}^{|V|}y_{t,j}\times log(\hat y_{t,j}) J(t)(θ)=−j=1∑∣V∣​yt,j​×log(y^​t,j​)

困惑度是评价语言模型好坏常用的方法，简称PPL,公式如下：

PP(S)=P(w1w2...wN)−1N=1p(w1w2...wN)N=∏i=1N1p(wi∣w1w2...wi−1)NPP(S)=P(w_1w_2...w_N)^{-\frac 1N}=\sqrt[N]{\frac {1}{p(w_1w_2...w_N)}}=\sqrt[N]{\prod_{i=1}^N\frac {1}{p(w_i|w_1w_2...w_{i-1})}}PP(S)=P(w1​w2​...wN​)−N1​=Np(w1​w2​...wN​)1​​=N∏i=1N​p(wi​∣w1​w2​...wi−1​)1​​

S代表句子，N是句子长度，p(wi)p(w_i)p(wi​)是第i个词的概率，所以上式的意思就是正常句子的概率越大，困惑度越小，语言模型越好。

2. LSTM

RNN缺点就在于短期记忆和容易梯度爆炸。从上图中我们可以看到,"历史信息"是会被作为输入不断地被计算的,也就意味着会被不断地稀释,而事实上经过3、4个神经节点之后,之前的信息基本上已经可以完全忽略了。

我们当然希望能记得越久越好,所以LSTM就应运而生。LSTM将上下文管理问题分成两个子问题:从上下文中移除不再需要的信息,以及添加对后续决策可能需要的信息。解决这两个问题的关键是学习如何管理这个上下文,而不是将策略硬编码到架构中。LSTM通过首先在架构中添加一个显式的上下文层(除了通常的循环隐藏层),并通过使用专门的神经单元实现这一点,这些单元使用门控制信息流入和流出构成网络层的单元。这些门通过使用附加的权重实现,这些权重顺序作用于输入、前一个隐藏层和前一个上下文层。

2.1 LSTM结构

LSTM的结构如下,下面我将分部分进行讲解:

(1) 状态保存层:

细胞节点状态的保存可以说是LSTM的关键所在。前面提到RNN的历史信息会随着计算迅速消失,那么为了解决这个问题,我们要不就干脆把 "历史状态"进行独立的存储。于是就出现了如图中"运输链"一般的结构,这使得信息能够长时间地保留。



(2) 遗忘门层:

该层的作用是:生成遗忘矩阵。

它对前一节点输出和当前节点输入进行sigmoid处理,这样就得到了一个值介于0~1的矩阵。0表示彻底遗忘,1表示完全保留。

(3) 输入门层:

该层的作用是:生成添加矩阵。

这一层由两个部分组成:

  • input gate layer: 通过sigmoid函数,生成(0~1)矩阵,表示将要更新哪些值。
  • tanh layer: 通过tanh函数,生成(-1~1)矩阵,表示新生成的候选值。

(4) 状态修改层:



该层的作用是:修改状态。

前面已经得到了遗忘矩阵和添加矩阵,那么现在只需要用状态矩阵乘以遗忘矩阵,就能遗忘相应内容;用状态矩阵加上添加矩阵,就能添加相应内容。这样状态既能长久保存,又能随着时间进行修改。

(5) 输出层:

输出层也分为两个部分:

  • input gate layer: 通过sigmoid函数,生成(0~1)矩阵,表示输入的特征分布。
  • tanh layer: 通过tanh函数,生成(-1~1)矩阵,表示已经学得的历史状态。

然后两者相乘,就得到了在历史中,遇到这种输入时的输出情况。

在传统的RNN中，梯度是通过时间逐步传递的，每一步都涉及权重的乘积，导致梯度可能随时间呈指数级衰减，从而造成梯度消失,使梯度不易趋于0。但在LSTM中，由于ftf_tft​的存在,梯度在一定程度上被“保护”起来，避免了这种指数级衰减。这里的ftf_tft​直接来自于遗忘门的激活，不涉及任何权重的乘积，也不依赖于过去的梯度值，它是当前输入和过去的隐藏状态的函数。

门控制机制允许网络学习何时“记住”或“忘记”某些信息，可以避免梯度爆炸或消失。
