引言：神经网络的起源与进化

神经网络作为自然语言处理（NLP）领域的一种基础计算工具，拥有深远的历史根基，其起源可追溯至1943年麦卡洛克和皮茨提出的简化生物神经元模型。这种模型最初旨在用命题逻辑的形式描述生物神经元的计算功能，因此得名“神经网络”。随着时间的推移，尽管现代语言处理的神经网络已不再直接依赖于这些初期生物学上的启示，但它们的基本构想和计算原理仍然保持不变，演化为由众多计算单元组成的网络，每个单元能够处理向量形式的输入并产生单一的输出值。

在语言处理领域中，特别是在进行分类任务时，神经网络的应用已经成为一种标准做法。所采用的架构通常是前馈网络，其特点是数据在网络中从一层单元传递到下一层单元，而不回流，这一计算模式保证了处理的有序性和可控性。随着深度学习的兴起，所谓的“深度”神经网络——即包含多个处理层的网络——已经成为研究的热点，它们因能够处理更复杂的数据结构和模式而受到青睐。

1. 神经网络概论

1.1 计算单元

在神经网络的基础研究中，计算单元的概念扮演着核心角色。每个计算单元接收一组实数作为输入，经过一定的计算过程后输出一个值。这种处理过程涉及到输入的加权求和，其中还包括了一个特殊的项，即偏置项。具体来说，对于输入向量 x1,x2,...,xnx_1, x_2, ..., x_nx1​,x2​,...,xn​ 和相应的权重 w1,w2,...,wnw_1, w_2, ..., w_nw1​,w2​,...,wn​，以及偏置项 bbb，可以表示为：

z=b+∑wixiz = b + \sum w_i x_i z=b+∑wi​xi​

1.2 前馈神经网络

前馈神经网络（Feedforward Neural Networks），也被称作多层感知器（MLP，Multi-Layer Perceptrons），构成了神经网络研究领域的基石之一。它们是一类多层结构的网络，在这些网络中，信息沿着单向路径前行，从输入层流向输出层，中间不经过任何循环或反馈路径。这种结构的一个显著特点是，每一层的输出仅作为下一层的输入，避免了信息的回流。

在早期，由于历史原因和对这类网络的初步理解，前馈网络有时被误称为“多层感知器”。实际上，现代的多层网络单元采用的是非线性函数，如Sigmoid，与最初的感知器模型（基于线性函数的简单单元）相去甚远。尽管如此，这一术语仍被广泛使用。

前馈神经网络主要包含三种节点：输入单元（Input Units），隐藏单元（Hidden Units），以及输出单元（Output Units）。如图下图所示，输入层（记作x）接收原始数据，这些数据通常以标量值的向量形式存在。隐藏层（记作h）是网络的核心，由若干隐藏单元（hih_ihi​）构成，每个单元根据上一小节的描述，对其输入执行加权求和操作后再应用一个非线性变换。在标准的前馈网络中，各层都是全连接的，意味着每一层中的单元都与前一层的所有单元相连。


对于隐藏层中的每个单元，其计算基于权重向量和偏置的参数,整个隐藏层的参数可以通过组合每个单元的权重向量和偏置形成单一的权重矩阵W和偏置向量b来表示。具体计算过程包括三个步骤：权重矩阵W与输入向量x相乘，加上偏置向量b，然后应用激活函数。因此，隐藏层的输出可以表示为：

h=σ(Wx+b)h = \sigma(Wx + b) h=σ(Wx+b)



输出层的作用是将隐藏层提供的新表征转换为最终输出。在分类任务中，这通常意味着将实数向量转换为概率分布，这一转换通过softmax函数完成，公式如下：

在整体上，前馈神经网络通过其简洁而强大的结构，在深度学习任务中扮演着关键角色。通过利用非线性激活函数和多层结构，这类网络能够捕捉和学习输入数据中的复杂模式，为各种预测和分类任务提供了一种有效的解决方案。

1.3 自然语言处理中的前馈网络应用

2. 神经网络的训练机制

前馈神经网络是一种监督机器学习的实例,其中我们知道每个观测值xxx的正确输出yyy。训练过程的目标是学习每层的参数,使得每个训练观测的y^\hat{y}y^​尽可能接近真实的yyy。

通常,我们通过在第5章为逻辑回归介绍的方法来完成这一切,因此读者在继续之前应该熟悉那一章。

完成目标首先需要一个损失函数来模拟系统输出与金标准输出之间的距离,通常使用逻辑回归中使用的损失函数——交叉熵损失。

其次,为了找到使这个损失函数最小化的参数,我们将使用梯度下降优化算法。

第三,梯度下降需要知道损失函数的梯度,即包含损失函数相对于每个参数的偏导数的向量。在逻辑回归中,对于每个观测,我们可以直接计算损失函数相对于个别www或bbb的导数。但对于有数百万参数和许多层的神经网络,计算某个第1层中某个权重的偏导数就更难了,因为损失是附加在一些更后面的层上的。我们如何在所有这些中间层上分配损失?答案是一个称为误差反向传播或反向微分的算法。

2.1 损失函数

在神经网络的培训过程中,交叉熵损失函数(负对数似然损失) 扮演了一个至关重要的角色,无论是在逻辑回归还是更复杂的多类别分类任务中。这一损失函数,其在二元分类中的形式为:

LCE(y^,y)=−log⁡p(y∣x)=−[ylog⁡y^+(1−y)log⁡(1−y^)]LCE(\hat{y},y) = -\log p(y|x) = -[y \log \hat{y} + (1-y) \log (1-\hat{y})] LCE(y^​,y)=−logp(y∣x)=−[ylogy^​+(1−y)log(1−y^​)]

扩展到多类别分类情境,交叉熵损失维持其核心原理不变,但适应性地调整以适合更广泛的应用场景。当面对超过两个类别的分类问题时,真实标签 y 和预测标签 y^\hat{y}y^​ 均被表示为向量。在这种情况下,对于单一观测 x,损失函数转化为所有 K 个输出类别的概率的负对数和,每一项都乘以其对应的实际概率 yky_kyk​:

LCE(y^,y)=−∑k=1Kyklog⁡y^kLCE(\hat{y},y) = -\sum_{k=1}^{K} y_k \log \hat{y}_k LCE(y^​,y)=−k=1∑K​yk​logy^​k​

特别地,当只有一个正确的类别时,真实标签 y 以one-hot向量的形式出现,且损失函数进一步简化为正确类别的输出概率的负对数:

LCE(y^,y)=−log⁡y^cLCE(\hat{y},y) = -\log \hat{y}_c LCE(y^​,y)=−logy^​c​

其中 c 是正确的类别。结合 softmax 函数,这表达式进一步演化为:

LCE(y^,y)=−log⁡exp⁡(zc)∑j=1Kexp⁡(zj)LCE(\hat{y}, y) = -\log \frac{\exp(z_c)}{\sum_{j=1}^{K} \exp(z_j)} LCE(y^​,y)=−log∑j=1K​exp(zj​)exp(zc​)​

这里,zcz_czc​ 是网络对于类别 c 的未归一化的预测值。

然而,深层网络的存在使得直接计算每个参数的梯度变得更加复杂,尤其是当参数分布在网络的多个层中时。为了有效地计算深层网络中的梯度,误差反向传播算法被提出。这一算法基于计算图的概念,允许梯度信息从输出层逆向传播至输入层,从而为每个权重提供更新方向，下节我们将来介绍计算图。




2.3 计算图与误差反向传播

计算图是计算数学表达式过程的表示,其中计算被分解为单独的操作,每个操作都被建模为图中的一个节点。

考虑计算函数L(a,b,c)=c(a+2b)L(a, b, c) = c(a + 2b)L(a,b,c)=c(a+2b)。如果我们使组成的加法和乘法操作显式,并为中间输出添加名称(d和e),则结果的一系列计算是:

d=2×bd = 2 \times b d=2×b

e=a+de = a + d e=a+d

L=c×eL = c \times e L=c×e

我们现在可以将其表示为一个图,其中节点用于每个操作,有向边显示从每个操作的输出作为下一个操作的输入。在下图中,我们假设输入a=3,b=1,c=−2a = 3, b = 1, c = -2a=3,b=1,c=−2,并且我们展示了前向传递计算结果L(3,1,−2)=−10L(3, 1, -2) = -10L(3,1,−2)=−10。


除了前向计算之外，计算图在执行反向微分时尤为重要，它用于计算损失函数相对于模型参数的导数，这对于基于梯度的优化方法至关重要。利用链式法则，反向微分从计算图的输出节点向输入节点逆向传播梯度，计算过程中涉及的每个节点都会根据其局部梯度和上游传来的梯度计算出下游梯度。这种方法不仅适用于简单的函数，也能高效地处理复杂的深度学习模型中的梯度计算。

反向微分的思想是从最终节点向所有图中的节点传递梯度。每个节点获取一个上游梯度,将其乘以局部梯度(其输出相对于其输入的梯度),并使用链式法则计算要传递给先前节点的下游梯度。如果一个节点有多个输入,则可能有多个局部梯度。

让我们现在计算我们需要的3个导数。因为在计算图中L=ceL = ceL=ce,我们可以直接计算导数∂L∂e\frac{\partial L}{\partial e}∂e∂L​:

∂L∂e=c,∂L∂c=e\frac{\partial L}{\partial e} = c, \frac{\partial L}{\partial c} = e ∂e∂L​=c,∂c∂L​=e

对于另外两个,我们需要使用链式法则:

∂L∂a=∂L∂e⋅∂e∂a\frac{\partial L}{\partial a} = \frac{\partial L}{\partial e} \cdot \frac{\partial e}{\partial a} ∂a∂L​=∂e∂L​⋅∂a∂e​

∂L∂b=∂L∂e⋅∂e∂d⋅∂d∂b\frac{\partial L}{\partial b} = \frac{\partial L}{\partial e} \cdot \frac{\partial e}{\partial d} \cdot \frac{\partial d}{\partial b} ∂b∂L​=∂e∂L​⋅∂d∂e​⋅∂b∂d​

在反向传递中,我们从右向左沿着图的每条边计算这些偏导数,使用链式法则正如我们上面所做的。因此,我们从节点L开始计算下游梯度,这些是∂L∂e\frac{\partial L}{\partial e}∂e∂L​和∂L∂c\frac{\partial L}{\partial c}∂c∂L​。对于节点e,我们然后将这个上游梯度∂L∂e\frac{\partial L}{\partial e}∂e∂L​乘以局部梯度(输出相对于输入的梯度),∂e∂d\frac{\partial e}{\partial d}∂d∂e​得到我们发送回节点d的输出:∂L∂d\frac{\partial L}{\partial d}∂d∂L​。依此类推,直到我们已经标注了图中所有输入变量的所有方式，下图展示了反向传递过程：







3. 神经网络在自然语言处理中的深度应用

3.1 前馈神经语言模型

在深入探讨前馈神经语言模型之前,重要的是要了解其在现代自然语言处理(NLP)中的核心地位。与传统的n-gram模型相比,神经语言模型提供了显著的优势,包括处理更长的历史信息能力、更好的上下文泛化能力以及更准确的词预测性能。然而,这些模型的复杂性、训练成本以及可解释性的挑战也不容忽视。尽管如此,对于需要捕捉长距离依赖和复杂语言模式的场景,神经语言模型成为了首选。

前馈神经语言模型通过接收一系列先前的词(如 wt−1,wt−2w_{t-1}, w_{t-2}wt−1​,wt−2​ 等)作为输入,并输出可能下一个词的概率分布,本质上近似了给定整个之前上下文中的一个词的条件概率 P(wt∣w1:t−1)P(w_t|w_1:t-1)P(wt​∣w1​:t−1)。这一过程可以通过公式简化表示为:

P(wt∣w1,...,wt−1)≈P(wt∣wt−N+1,...,wt−1)P(w_t|w_1,...,w_{t-1}) \approx P(w_t|w_{t-N+1},...,w_{t-1}) P(wt​∣w1​,...,wt−1​)≈P(wt​∣wt−N+1​,...,wt−1​)

其中, N−1N-1N−1 表示考虑的先前词的数量。

神经语言模型利用词嵌入代替仅使用词身份的方法,这允许模型在未见数据上实现更好的泛化。这种泛化能力的一个示例是模型能够基于"cat"和"dog"有相似嵌入的知识,从一个上下文泛化到另一个上下文,即使它从未见过"dog gets fed"这样的短语。


3.2 神经语言模型的前向推断

在神经语言模型中进行前向推断是一个将给定输入转化为可能输出(即下一个词)的概率分布的过程。这一过程从将N个先前的词各自表示为长度为|V|的独热向量开始,其中|V|是词汇表的大小,独热向量在与词对应的词汇索引的维度上值为1,其余维度为0。

例如,考虑一个3-gram模型,我们通过嵌入矩阵EEE将这些独热向量转换为词嵌入,嵌入矩阵的维度为d×∣V∣d \times |V|d×∣V∣,其中d是嵌入的维度。通过与独热向量的乘积,我们可以从EEE中选择与特定词对应的嵌入向量。将这些嵌入向量连接起来,我们得到嵌入层eee,接着是隐藏层和输出层,输出层的softmax函数产生了词汇表上的概率分布。

这个过程可以通过以下算法详细描述:

选择嵌入: 对于三个先前的词,我们查找它们在嵌入矩阵EEE中的嵌入,形成嵌入层eee。

e=[Et−3;Et−2;Et−1]e = [E_{t-3}; E_{t-2}; E_{t-1}] e=[Et−3​;Et−2​;Et−1​]

计算隐藏层: 嵌入层通过权重矩阵WWW(加偏置bbb)并应用激活函数(如ReLU)来转换为隐藏层hhh。

h=σ(We+b)h = \sigma(We+b) h=σ(We+b)

计算输出层: 隐藏层通过另一个权重矩阵UUU转换,并应用softmax函数得到下一个词的概率分布。

z=Uhz = Uh z=Uh

y^=softmax(z)\hat{y} = \text{softmax}(z) y^​=softmax(z)


通过这种方式,神经语言模型能够有效地利用过去的信息来预测下一个词,从而在捕捉语言的复杂性方面超越了传统的n-gram模型。

3.3 神经语言模型的训练

神经语言模型的训练基于自监督的概念，其训练的高层思路都是利用文本语料库作为训练材料，通过模型在每个时间步骤t预测下一个词来进行训练。由于每个时间步的正确答案（即语料库中的下一个词）是已知的，因此可以通过最小化预测错误来训练模型以更准确地预测下一个词。

训练模型（包括嵌入层E）涉及通过梯度下降方法设置所有参数θ={E,W,U,b}\theta = \{E, W, U, b\}θ={E,W,U,b}，并利用计算图上的误差反向传播算法计算梯度。这种训练方法不仅涉及调整网络的权重W和U，而且在预测即将出现的词汇的同时，也学习了最能预测下一词的词嵌入E。

下图展示了设置窗口大小为N=3的上下文词的配置。输入x由3个独热向量组成，通过3个实例化的嵌入矩阵E完全连接到嵌入层。这里不希望为映射每一个前置词到投影层学习独立的权重矩阵，而是希望有一个共享的嵌入字典E。这样，无论哪个词出现在wt−2w_{t-2}wt−2​或wt−1w_{t-1}wt−1​的位置，都用同一个向量来表示。嵌入权重矩阵E对每个词有一个列向量，每个列向量有d维，因此E的维度为d×∣V∣d \times |V|d×∣V∣。

训练过程通常从输入一段很长的文本开始，将所有句子连接起来，从随机权重出发，然后迭代地遍历文本预测每个词wtw_twt​。在每个词wtw_twt​处，采用交叉熵损失来评估模型的性能。这个损失的公式为：

LCE(y^,y)=−log⁡y^iLCE(\hat{y}, y) = -\log \hat{y}_i LCE(y^​,y)=−logy^​i​

其中iii是正确类别。对语言建模而言，类别是词汇表中的词，所以y^i\hat{y}_iy^​i​代表模型赋予正确下一个词wtw_twt​的概率。随机梯度下降(SGD)的参数更新公式为：

θs+1=θs−η∂[−log⁡p(wt∣wt−1,...,wt−n+1])∂θ\theta_{s+1} = \theta_s - \eta \frac{\partial [-\log p(w_t|w_{t-1},...,w_{t-n+1}])}{\partial \theta} θs+1​=θs​−η∂θ∂[−logp(wt​∣wt−1​,...,wt−n+1​])​

此梯度可以在任何标准神经网络框架中计算，并通过θ={E,W,U,b}\theta = \{E, W, U, b\}θ={E,W,U,b}进行反向传播。通过最小化损失来训练参数，不仅能得到一个用于语言建模的算法（即词预测器），还能得到一套可以用于其他任务的新词嵌入E。


4.问题

在下面提及的词嵌入方法中，尽管它们在泛化能力上相较于n-gram模型有所增强，但每个词语仅被分配一个单一的向量。对于“mouse”这样的多义词，词嵌入方法是如何区分其不同含义呢？

以句子“I have to make sure that the cat gets fed.”为例，词语"cat"和"gets"之间的相互影响对理解整个句子是有帮助的。然而，对于像“the”这种影响较小的词语(它对预测下一个词的出现几乎无帮助)我们应当如何处理呢?将这种类型的词语在模型训练的时候去掉是否对训练有帮助呢？
